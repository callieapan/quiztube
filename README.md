# QuizTube

Welcome to QuizTube, an innovative application designed to enhance your YouTube viewing experience by summarizing videos, generating quiz questions, and providing answers. Whether you're a student, educator, or just a curious mind, QuizTube helps you better engage with video content.

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Features

- **Video Summarization**: Automatically generates a concise summary of any YouTube video.
- **Quiz Generation**: Creates a set of questions based on the video content to test your understanding.
- **Answer Provision**: Supplies answers to the generated quiz questions for self-assessment.
- **User-Friendly Interface**: Easy to use, simply input a YouTube URL and let QuizTube do the rest.

## Installation

To get started with QuizTube, follow these steps:

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/yourusername/QuizTube.git
    ```
2. **Navigate to the Project Directory**:
    ```bash
    cd QuizTube
    ```
3. **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. **Run the Application**:
    ```bash
    python app.py
    ```
2. **Input YouTube URL**:
    - Open your browser and navigate to `http://localhost:5000`.
    - Enter the YouTube URL of the video you want to summarize.
3. **View Summary and Quiz**:
    - The app will display a summary of the video.
    - It will also generate quiz questions with corresponding answers based on the video content.
